How it works

This demo shows how to integrate the Patient Search library into any existing HTML page. The library creates a modal overlay that pops up when triggered.

Integration Code

To add this to your page, just include the script and initialize:

<script src="patient-search.js"></script>
<script>
  const search = new PatientSearch({
    onSelect: (patient) => {
      console.log('Selected:', patient);
      // Handle the selected patient
    }
  });
  
  // Show the modal when needed
  document.getElementById('myBtn')
    .addEventListener('click', () => {
      search.show();
    });
</script>
