Iframe Integration Demo

How it works

This demo shows how to integrate the Patient Search library using an iframe. The library runs inside the iframe and communicates with the parent page using postMessage.

üí° Tip: The iframe loads the patient search interface and sends the selected patient data back to this parent page using window.postMessage(). This approach is useful when you need complete isolation between the search interface and your main application.

Integration Code

To integrate via iframe:

<!-- Parent Page -->
<iframe id="patientSearchFrame" src="iframe-content.html"></iframe>

<script>
  // Listen for patient selection from iframe
  window.addEventListener('message', (event) => {
    if (event.data.type === 'patientSelected') {
      const patient = event.data.patient;
      console.log('Selected:', patient);
      // Handle the patient data
    }
  });
</script>
